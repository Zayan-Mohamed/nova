### Search mode (after pressing `/`)

| Key               | Action                                    |
| ----------------- | ----------------------------------------- |
| _type characters_ | Filter networks in real time              |
| `Enter`           | Apply filter and exit search mode         |

| `Ctrl+U`          | Clear the current search input            |
| `Backspace`       | Delete last character                     |
